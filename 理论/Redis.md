```
Redis 是一个开源的使用 ANSI C语言编写、遵守 BSD 协议、支持网络、可基于内存、分布式、可选持久性的键值对(Key-Value)存储数据库，并提供多
种语言的 API
```

# 简介

```
 特点：********
     1. 数据的持久化。内存的数据可以保存在磁盘，重启的时候可以再次加载进行使用

     2. 支持k-v类型数据、list，set，zset，hash等数据结构的存储

     3. 数据的备份。master-slave模式的备份


```

```
 优势：
     1. 性能极高

     2. 数据类型丰富。 支持Strings，Lists, Hashes, Sets, Ordered Sets

     3. 原子。 所有操作都是原子的。多操作也支持事务

     4. 丰富的特性。 


 不同之处
     1. 数据结构更复杂，也有对应的原子性操作

     2. 内存运行，但可以持久化到磁盘
```
